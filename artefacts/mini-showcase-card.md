# Mini Showcase Card

Use this for the 7:15 PM - 7:25 PM sharing section.

## Format

Each person gets 1-2 minutes.

Ask them:

1. What did you start with?
2. What did you ask Codex?
3. What did Codex produce?
4. What did you accept, change, or reject?
5. What is your next step?

## What To Look For

Try to include:

- One Recipe Finder example
- One product or startup example
- One docs, copy, or README example
- One developer example

## Closing Prompt

After each share, ask:

```text
What would you ask Codex next?
```

## Final Wrap Line

> The pattern is the same across all examples: idea, plan, small change, review, next step.
