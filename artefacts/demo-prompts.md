# Recipe Finder Demo Prompts

These prompts support the live demo.

## Messy Idea

```text
I want to build a simple recipe website for busy people who want to find meals based on ingredients they already have.
Help me clarify the product idea.
```

Expected output:

- Target user
- Problem
- MVP
- Feature ideas
- First task

## Small Plan

```text
Turn this into a small implementation plan.
```

Expected output:

- Pages or components
- Data structure
- Smallest useful feature
- Steps

## Starter Project Inspection

```text
Inspect this starter recipe website.
Explain how it is structured and what would be the smallest useful improvement.
Do not change files yet.
```

Expected output:

- File map
- Existing behavior
- Small improvement options
- Recommended first change

## Ingredient Search Feature

```text
Inspect this starter recipe website.
Add a simple ingredient search filter.
Keep the change small and explain what files you changed.
```

Expected output:

- Search input
- Filtered recipe cards
- Clear explanation of changed files

## Review

```text
What did you change?
What should I review carefully?
What could break?
```

Expected output:

- Change summary
- Review checklist
- Risk areas

## Handoff

```text
Write a short summary of what changed, how to test it, and what to improve next.
```

Expected output:

- Summary
- Test steps
- Next ideas

## Participant Practice Prompt

```text
I want to use the same loop on my own topic.
Help me clarify the idea, propose a small plan, improve one thing, review the output, and summarise the next step.
```
